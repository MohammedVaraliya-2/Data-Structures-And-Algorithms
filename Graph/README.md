### 01. Graph Data Structure

    Graph Data Structure
    

    Example Graph:

                  />  Paris
                 /     |     \  
                /      |      \
               /       |       \
              /        |        \> New York       
        Mumbai         |         />         \ 
              \        |        /            \
               \       |       /              \
                \      |      /                \
                 \     |>    /                  \> Toronto
                  \>  Dubai   


    The Routes:

        ("Mumbai", "Paris"),
        ("Mumbai", "Dubai"),
        ("Paris", "Dubai"),
        ("Paris", "New York"),
        ("Dubai", "New York"),
        ("New York", "Toronto")
